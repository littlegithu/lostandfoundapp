## Lost & Found App

>A React-based web application that helps users manage lost and found items. Users can add, edit, and view items, as well as manage their profile with a clean and responsive UI.

## Features
>Home page to view items
>Add new lostandfound items
>Edit existing items
>User profile management
>Light/Dark theme support
>Responsive layout with sidebar and footer
>Client-side routing with React Router
>Global state management using Context API

## Tech Stack
>Frontend: React.js (Functional Components & Hooks)
>Routing: react-router-dom
>State Management: Context API
>hemeContext
>UserContext
>Styling: Tailwind CSS
